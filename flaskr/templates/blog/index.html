Pathtracer

[C++]

I implemented core functionality for a 3D scene renderer. Methods included ray tracing, Monte Carlo Integration, and light transport via BSDF's. Optimization techniques to decrease rendering time include Bounding Volume Heirarchy Acceleration and Adaptive Sampling.


Stock price

[python]

A web scraper I created to retrieve current stock prices. It takes a stock tick symbol as input and returns the current day's high, low, open, and close. Powered by the BeautifulSoup4 python library, it scrapes the desired data from Yahoo Finance, and is built on a Tkinter GUI.

Cloth Simulator

[C++]

I implemented a real-time cloth simulator by modeling cloth material as a spring-mass system. I built the data structures which represent the cloth and the constraints that define the behavior. Verlet Integration was employed to derive cloth particles updated positions at each new timestep. Finally, various shaders were written including Blinn-Phong, Displacement Mapping, and Environment-mapped Reflections.


Volumetric Scattering

[C++]

I designed and developed a volumetric scattering renderer to simulate godrays. I used a ray marching approach combined with a decaying property associated with radiance/space.
