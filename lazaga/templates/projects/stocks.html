Page Fetching


I examined various Yahoo Finance stock quote pages to identify patterns. Luckily, the site path leading to each particular stock quote is easily located by appending the stock tick symbol to the end of the path, e.g., https://finance.yahoo.com/quote/GME/ for GME stock, https://finance.yahoo.com/quote/GOOGL/ for GOOGL stock, etc.

Data Parsing


I used the BeautifulSoup4 python library to robustly parse all of the text data on the stock quote page in question. I identified patterns that reflect whether the stock tick symbol is valid or invalid. Invalid queries result in an error message being displayed, while valid queries then parse the current calendar day's stock quote, particularly the high, low, open, and close figures.
