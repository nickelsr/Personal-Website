Cloth Simulator

Spring-Mass System


Structural, bending, and shearing springs connect the point masses that comprise the cloth to create a realistically maleable surface.

Simulation


Simulation of cloth physics is accomplished by updating point mass positions for each frame via Verlet integration, which accounts for external forces (e.g., gravity) and internal forces (springs connecting neighboring point masses).

Collision Handling


Behavior is defined for cloth point masses colliding with spheres and planes, as well as self-collisions, increasing the situational simulation opportunities.

Shading


Finally, to create more rich and realistic textures and color schemes to our cloth, various shaders are written in GLSL which can be attached to the cloth.
